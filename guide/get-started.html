⚙️使用组件 #

【推荐】自动按需引入 #

TIP

配合unplugin-vue-components实现自动按需引入，免去了全局注册或者局部注册的繁琐，同时也不需要考虑按需引入。

unplugin-vue-components 能够自动扫描模板中的组件标签，根据 resolver 按需自动配置注册组件，并生成对应的 d.ts 类型。

  • vite.config.ts配置如下：
js
import Components from "unplugin-vue-components/vite";

export default defineConfig({
  plugins: [
    Components({
      resolvers: [
        (componentName) => {
          // where `componentName` is always CapitalCase
          if (/^J[A-Z]/.test(componentName)) {
            return {
              name: componentName.slice(2),
              from: "jview-ui",
              as: componentName,
            };
          }
        },
      ],
    })
  ],
  css: {
    preprocessorOptions: {
      less: {
        javascriptEnabled: true,
      },
    },
  },
});

局部注册 #

以Button组件为例说明。

  • 首先引入并局部注册组件：
js
import { defineComponent } from 'vue';
import { Button } from "jview-ui";

export defineComponent({
  components: {
    Button
  }
})

全局注册 #

【推荐】可以全局注册某个组件

js
import { createApp } from "vue";
import { Button } from "jview-ui";

const app = createApp();

// 全局注册 Button 组件
app.use(Button);
